Práctica 13

Busqueda de número:
Crea una aplicación web que permita al usuario buscar un número específico dentro de una lista predefinida de números. La interfaz de usuario incluirá un input de números y un botón "Buscar". El usuario ingresará un número en el input y la aplicación verificará si el número ingresado se encuentra en la lista. Si se encuentra, la aplicación mostrará un mensaje indicando que el número se encontró en la lista. Si no se encuentra, mostrará un mensaje indicando que el número no se encontró en la lista.
